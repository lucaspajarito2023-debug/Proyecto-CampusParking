# 🚗CAMPUS-PARKING🏍️
#### _DESCRIPCION:_
Campus parking es un proyeccto de gestion de vehiculos en un estacionamiento, calculo de el tiempo de permanencia y el costo del servicio prestado. Ya que actualmente se estaban realizando el procedimineto de manera manual e implicaba perdida de clientes y disgustos entre los clientes. 


# 🛠️Funcionamientos principales
1. Login.
2. Visualizacion del perfil: Pertmite realizar cambios en datos.
3. Registrar entradas.
4. Ver slots.
5. Finalizar servicios.
6. Cerrar sesion.
